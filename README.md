# 🤖 Rule-Based AI Chatbot

A simple Rule-Based AI Chatbot developed using Python as **Project 1** for the **DecodeLabs Artificial Intelligence Internship**.

This chatbot interacts with users by responding to predefined questions using rule-based logic. It demonstrates the fundamental concepts of Artificial Intelligence, control flow, decision-making, and Python programming without using Machine Learning or external AI libraries.

---

## 📌 Project Overview

The Rule-Based AI Chatbot is designed to simulate basic human conversation using predefined responses stored in a Python dictionary.

The chatbot continuously accepts user input, processes it, and returns the appropriate response based on predefined rules.

This project focuses on understanding the core concepts of Artificial Intelligence before moving toward Machine Learning and Deep Learning.
